Flutter conversion of the React project in this workspace.

Quick start:
1. Open a terminal in flutter_port
2. Run: flutter pub get
3. Run: flutter run

Implemented screens:
- Home (projects)
- Project board
- Task detail
- Team roster
- Deadlines

Notes:
- Uses in-memory mock data matching the React app.
- Bottom navigation routes to Home, Team, and Deadlines.
